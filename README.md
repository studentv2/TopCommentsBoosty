# Top Comments (Boosty) — Расширение для сбора популярных комментариев

Расширение автоматически разворачивает все комментарии на Boosty, собирает топ комментариев по количеству реакций и выводит их в удобной таблице. Также отдельно отображаются изменённые комментарии (которые нарушают правило «не менять комментарий»).

---

## Возможности
- Автоматическое разворачивание всех комментариев на странице.
- Сбор комментариев с реакциями и сортировка по количеству лайков.
- Вывод топ-комментариев (4 и более с одинаковыми лайками, если есть совпадения).
- Отдельная таблица для изменённых комментариев.
- Подстройка таблицы под тёмную/светлую тему сайта.
- Кнопка копирования результата в буфер обмена.
- Не зависит от ручного вставления скрипта в консоль.

---

## Установка (для Chrome / Edge)
1. Клонируйте репозиторий или скачайте ZIP:
   ```bash
   git clone https://github.com/<username>/<repo>.git

2. Откройте chrome://extensions (или edge://extensions).

3. Включите Режим разработчика.

4. Нажмите Загрузить распакованное расширение и выберите папку с расширением.

5. На посте Boosty откройте панель расширений и нажмите кнопку Запустить сбор.
